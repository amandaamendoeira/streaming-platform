🎬 Streaming Platform — Profile Selector UI

A responsive profile selection interface inspired by modern streaming platforms like Netflix.
This project replicates the user profile selection screen commonly seen in streaming services, focusing on accessibility, responsive design, and theme switching.

Built during the Alura Front-End & AI Immersion, with emphasis on clean UI and user experience.

✨ Features

- Multiple user profile selection interface;
- Dark / Light mode with persistent preference (localStorage);
- Fully responsive layout (mobile, tablet, desktop)
- Netflix-inspired modern UI
- Hover interactions and visual feedback
- Accessibility-friendly structure (semantic HTML + ARIA)
- Lightweight and fast — no frameworks

🧠 Built With
- HTML5 — Semantic structure and accessibility
- CSS3 — Flexbox, media queries, custom properties (CSS variables)
- JavaScript (Vanilla) — Theme toggle and persistence
- LocalStorage API — User preference saving
